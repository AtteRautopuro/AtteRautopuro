Git ja GitHub

Etusivu

Git

Git on hajautettu versionhallintajärjestelmä, jota käytetään ohjelmistokehityksessä seuraamaan ja hallitsemaan muutoksia koodissa. Se mahdollistaa kehittäjien yhteistyön eri paikoissa, koska kaikki voivat työskennellä paikallisesti omilla kopioillaan ja yhdistää muutokset myöhemmin. Gitin avulla voi tarkastella ja palauttaa aiempia versioita, seurata erillisiä kehityslinjoja (haaroja) ja hallita useiden kehittäjien tekemiä päivityksiä samanaikaisesti ilman konfliktia.

GitHub

GitHub on pilvipohjainen palvelu, joka käyttää Git-versionhallintaa projektien hallintaan ja yhteistyöhön. Se tarjoaa alustan, jossa kehittäjät voivat tallentaa, jakaa ja hallita koodia sekä tehdä yhteistyötä tiimien kanssa. GitHub tukee myös avoimen lähdekoodin projekteja, pull requesteja, bugiraportteja ja jatkuvaa integrointia (CI). Se on suosittu työkalu ohjelmistokehityksessä, ja sen avulla tiimit voivat työskennellä samanaikaisesti samassa projektissa helposti ja hallitusti.
